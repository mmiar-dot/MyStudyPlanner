# MyStudyPlanner v1.5.0 - Notes de Release

## 🔔 Nouvelles fonctionnalités

### Système de Notifications
Recevez des rappels quotidiens pour ne jamais manquer vos sessions de révision !

- **Notifications des sessions du jour** : Un rappel chaque jour avec la liste de vos sessions à réviser
- **Notifications des sessions en retard** : Une alerte pour les révisions que vous avez manquées
- **Heure personnalisable** : Choisissez l'heure qui vous convient (7h, 8h, 9h, 12h, 18h, 20h ou heure personnalisée)

### Comment activer les notifications ?
1. Allez dans **Profil** > **Notifications**
2. Activez les types de notifications souhaitées
3. Choisissez l'heure de rappel
4. C'est tout ! Vous recevrez vos rappels à l'heure choisie

> ⚠️ **Note** : Les notifications push fonctionnent uniquement sur l'application mobile (iOS/Android). La version web affiche un message explicatif.

## 🚀 Comment mettre à jour ?

### iOS (TestFlight)
1. Ouvrez l'app TestFlight
2. Trouvez MyStudyPlanner
3. Appuyez sur "Mettre à jour"

### Android
1. Téléchargez l'APK depuis les releases GitHub
2. Installez la mise à jour

### Desktop (macOS/Windows)
- L'app se met à jour automatiquement au démarrage
- Ou téléchargez la dernière version depuis les releases GitHub
