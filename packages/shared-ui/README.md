# @mystudyplanner/shared-ui

Composants React Native partagés entre les applications mobile et desktop.

## Installation

Ce package est utilisé en interne dans le monorepo.

## Composants disponibles

- `ColorPicker` - Sélecteur de couleur
- `MethodSelector` - Sélecteur de méthode de révision
- `ProfilePhotoManager` - Gestion de la photo de profil
- `SRSRatingModal` - Modal de notation SRS
- `SessionCard` - Carte de session d'étude
- `StatsDetailModal` - Modal de détail des statistiques

## Utilisation

```tsx
import { SessionCard, ColorPicker } from '@mystudyplanner/shared-ui';
```
